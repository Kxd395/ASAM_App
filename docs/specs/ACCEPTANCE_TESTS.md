# Acceptance tests

- Two minute completion path with minimal required fields yields a PDF under size limit.
- Safety banner path: set blocker, record action, export unblocks.
- Six ratings required: export cannot proceed with any missing rating.
- WM indicated requires WM set or rationale documented.
- Discrepancy reasons required when levels differ.
- Dynamic renderer: 1, 7, 23 substance rows render with correct pagination and footers.
- EMR failure does not block assessment. Drawer shows error state.
- Idle lock hides PHI and blanks thumbnails.
